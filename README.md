# Deck Of Cards Program
Java Assignment 4 <br>
Rank = ace, 2, 3, 4, 5, 6, jack, king... (conceptually not tied to 13)<br>
Suit = hearts, spade... (conceptually not tied to 13)<br>

These are the three main classes of this program:

<h1>1. Card() </h1>
This class is the parent class for all remaining classes, deck of cards is built primarily from this class. It creates Card objects  
<h1>2. DeckOfCards() </h1>
A very complex class that lets me create deck out of a user inputting into the instance variables "maxRank" and "numberOfSuits". 
<h1>3. Main() </h1>
This is the view class and does most of the calling.

<h1>For using//demoing </h1>
Any JDK works, no JavaFX setup needed, UML diagrams also added and can be opened in draw.io<br>Otherwise, just open the three .java files :)
